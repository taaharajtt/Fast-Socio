# Page 14 --- Profile Specification

## Purpose

The Profile is the user's digital identity containing personal
information, achievements, activity, reputation, and social presence.

## Sections

Header, avatar, cover photo, bio, department, semester, Aura, XP, level,
badges, achievements, interests, societies, languages, availability,
prompt answers.

## Tabs

Posts, media, comments, saved, events, achievements, badges, activity,
statistics.

## Actions

Edit profile, follow, message, invite to event, report, block, share
profile, copy profile link, privacy settings.

## Backend

Profile completeness calculation, privacy enforcement, statistics
aggregation, activity timeline, cached counters, verification status,
media management.

## Navigation

Profile → Edit Profile / Settings / Followers / Following / Event
History / Achievements
