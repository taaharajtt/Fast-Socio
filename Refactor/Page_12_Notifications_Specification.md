# Page 12 --- Notifications Specification

## Purpose

Centralize every important activity while preventing notification
fatigue.

## Types

Likes, comments, replies, mentions, follows, matches, messages, event
reminders, registrations, leaderboard movement, Aura gains,
achievements, badges, society announcements, system alerts, security
alerts, onboarding reminders.

## Functions

Grouped notifications, mark as read, mark all read, delete, mute
categories, deep linking, push notifications, in-app notifications,
scheduled reminders, digest summaries.

## Backend

Notification queue, delivery retries, read state synchronization,
priority levels, batching, deduplication, expiration rules, analytics.
