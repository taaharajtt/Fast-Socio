# Brand assets

Drop the home-page logo here as **`logo.png`** (i.e. `public/brand/logo.png`).

- The home header renders it via `next/image` at `h-7` (28px tall), width auto.
- A transparent-background PNG works best; any aspect ratio is fine (it scales to height).
- The literal path referenced in code is `/brand/logo.png` — keep that filename.
